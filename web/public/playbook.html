Table Top Exercise

Summary

An eager intern is distraught to learn their favorite note taking app, Obsidian, is internally banned. Frothing with rage they take to The Google and search for the next best solution. In their frustration they click the first result, a poisoned search result that leads them to a malicious version of a popular markdown notes app called Beaver Notes.

Narratives

Thinking about this from multiple angles, I've formulated different narratives that describe the goals of the table top exercise.

The Concerns

  • Unreviewed/unapproved software.
  • The depth and accuracy of software reviews.
  • Malicious search results.
  • Can we RE? :D

The Threat Actor (Void Serpent)

  • Would we detect low privileged esionage tradecraft.
  • Would we catch a threat actor leveraging tactics to masquerade as a disgruntled employee?
  • Do we have visibility into websockets.

Attack Chain

  1. Malicious Google Search advertisement.
    • T1583.008 Malvertising
  2. User unprivileged install of unapproved software.
    • T1587.001 Malware?
    • T1587.002 cloned code signing certs.
  3. Scheduled task persistence "Beaver Update".
    • T1503.005 Scheduled Task Persistence.
  4. WebSocket C2 communication.
    • T1071.001 Application Layer Protocols.
  5. System User Discovery
    • T1033 System Owner/User Discovery
  6. Active directory enumeration.
  7. MRU documents enumeration and exfiltration.
  8. Secondary stage downloads.
  9. Forced authentication through .lnk/library-ms files.
    • T1187 Force Authentication.
    • Reference: New Windows NTLM Leak Flaw
